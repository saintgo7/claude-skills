# gem-llm-troubleshooting

GEM-LLM 운영 중 발생하는 13개 알려진 에러 사례 빠른 참조 + 해결책. 책 16장 case 1~13의 압축본.

## 사용 시점

- "에러", "안 돼", "500", "디버그"
- "OOM", "disk I/O", "QueuePool"
- "unknown_model", "Connection refused", "layer_scalar"
- "vLLM 죽었어", "왜 느려"

## 설치

```bash
./install.sh gem-llm-troubleshooting
```

빠른 진단 흐름과 케이스별 해결법은 [SKILL.md](SKILL.md) 참조.
