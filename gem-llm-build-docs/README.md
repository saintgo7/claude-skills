# gem-llm-build-docs

GEM-LLM 프로젝트의 책/매뉴얼/논문(한/영) 빌드를 책임지는 스킬. Pandoc + XeTeX 파이프라인.

## 사용 시점

- "gem-llm 문서 빌드", "book-ko 빌드"
- "paper-en 빌드", "manual-en 빌드"
- "diagrams 렌더", "all docs"
- "mermaid SVG 변환"

## 설치

```bash
./install.sh gem-llm-build-docs
```

`make book-ko / paper-en / diagrams / all / clean / doctor` 명령 매핑은 [SKILL.md](SKILL.md) 참조.
