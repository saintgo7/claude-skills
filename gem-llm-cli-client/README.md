# gem-llm-cli-client

`gem-cli` (Claude Code 스타일 터미널 클라이언트) 사용법 + 디버깅 스킬.

## 사용 시점

- "gem CLI", "gem login", "gem chat"
- "REPL", "슬래시 명령", "/cost", "/model 변경"
- "tool calling 안됨", "gem-cli 설치"
- "auto-run"

## 설치

```bash
./install.sh gem-llm-cli-client
```

config 파일 (`~/.config/gem-cli/config.json`) 위치, 사용자 머신/n1 동일 동작은 [SKILL.md](SKILL.md) 참조.
