# searcam-book skill

SearCam 개발기 기술 서적의 챕터를 한국어·영어로 동시에 작성하는 Claude Code 슬래시 커맨드.

## 설치

```bash
cp searcam-book.md ~/.claude/commands/
```

Claude Code 재시작 후 `/searcam-book` 으로 사용.

## 사용법

```
/searcam-book "Ch11 Wi-Fi 스캔 챕터 작성"
/searcam-book "Ch18 테스트 챕터 — 새 테스트 케이스 반영"
/searcam-book --list
```

## 파일

- `searcam-book.md` — 슬래시 커맨드 정의
