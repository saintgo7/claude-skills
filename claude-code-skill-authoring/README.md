# claude-code-skill-authoring

Claude Code skill을 작성하는 메타 skill — 32+ skill 작성에서 검증된 패턴.

## 사용 시점

- "skill 만들기" / "SKILL.md 작성" / "트리거 phrase"
- "claude-skills 추가" / "skill scaffolding" / "skill 검증"

## 설치

```bash
./install.sh claude-code-skill-authoring
```

## scaffold.sh로 새 skill 부트스트랩

```bash
bash claude-code-skill-authoring/scripts/scaffold.sh my-skill "한 줄 설명"
```
