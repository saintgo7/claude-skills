# gem-llm-overview

GEM-LLM 프로젝트 전체 구조와 컴포넌트 위치를 빠르게 파악하는 스킬. 시스템에 새로 합류하거나 어떤 컴포넌트가 어디 있는지 확인할 때 사용.

## 사용 시점

- "gem-llm이 뭐야", "전체 구성"
- "어디서 시작해", "프로젝트 구조"
- "포트 매핑", "어떤 모델 사용"
- "llm.pamout.com 어디로 가"

## 설치

```bash
./install.sh gem-llm-overview
```

깊은 디버깅은 다른 `gem-llm-*` skill로. 컴포넌트 토폴로지는 [SKILL.md](SKILL.md) 참조.
