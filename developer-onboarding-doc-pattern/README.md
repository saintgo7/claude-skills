# developer-onboarding-doc-pattern

신규 개발자 0 → 실행 onboarding 문서 묶음 표준 패턴.
README + QUICKSTART + INDEX 4-tier 구조 + 5분 검증 단계 + FAQ + 트러블슈팅.
gem-llm `README.md` / `INDEX.md` / `QUICKSTART.md` / `docs/ops/*` 검증.

## 사용 시점

- 신규 입사자 첫 주 onboarding
- 새 환경 setup 가이드 (`git clone` → first run)
- 외부 contributor 첫 PR 가이드
- 인수인계 문서

## 설치

```bash
./install.sh developer-onboarding-doc-pattern
```

## 빠른 시작

```bash
cp ~/.claude/skills/developer-onboarding-doc-pattern/templates/README.md.template README.md
cp ~/.claude/skills/developer-onboarding-doc-pattern/templates/QUICKSTART.md.template QUICKSTART.md
```
