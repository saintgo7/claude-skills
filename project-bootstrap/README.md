# project-bootstrap

bilingual research/code 프로젝트를 한 번에 자동 부트스트랩하는 스킬. GitHub repo + Cloudflare 서브도메인 + 한/영 책 골격 + 한/영 논문 템플릿 + Mermaid 다이어그램까지.

## 사용 시점

- "새 프로젝트 시작", "프로젝트 부트스트랩/init"
- "한영 책 같이", "논문 템플릿 자동"
- "GEM-LLM 같은 구조로 새로 시작"
- "research-project init"

## 설치

```bash
./install.sh project-bootstrap
```

인자 (프로젝트 이름 + 선택 도메인), master 노드 cloudflared/gh 자격증명 (saintgo7) 자동 사용은 [SKILL.md](SKILL.md) 참조.
